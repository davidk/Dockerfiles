Instructions:

`docker build -f f3.dockerfile -t f3 .`

Mount the removable drive onto your filesystem somewhere.

Running

	 docker run --privileged \
	 -v /replace/with/path/to/removable_media:/f3 \
	 f3 \
	 bash -c 'f3write /f3 && f3read /f3'
